### 增加网络的稀疏度
+ 添加Dropout
	+ 随机使部分神经元不工作
	+ 测试的时候使用全部神经元, 并对权重进行处理(例如dropout 是 0.7 即 70% 的神经元不工作, 测试的时候要将该层的权重变为 3/10), 因为测试的时候整体神经元数量较多, 权重不变的情况下, loss会变大, 为保证梯度不变, 需要将权重减小
